# FortiView

<img width="558" height="314" alt="image" src="https://github.com/user-attachments/assets/be6256d2-0250-45ed-b160-02d9666a52c2" />

O FortiView exide dados em tempo real e históricos, oferecendo diversos painéis para fornecer visão resumida das informações da sua rede. O FortiView exibe dados de logs de análise, no entanto, dados de logs arquivados não são exibidos no FortiView.

Cada ADOM possui sua própria visualização de análise de dados no FortiView. Ao visualizar os dados de um ADOM, certifique-se sempre de estar no ADOM correto. 

Pode-se desativar o módulo FortiView do FortiAnalyzer quando o necessário para otimizar o desempenho, usando os comandos:

```bash
#config system global
    set disable-module fortiview-noc
end
```
Quando desativado, a interface gráfica oculta o FortiView e interrompe o processamento em segundo plano desse recurso. 

# Dashboards do FortiView

<img width="563" height="317" alt="image" src="https://github.com/user-attachments/assets/ca07a5a1-0a57-4a3f-bf0d-6e023f794b35" />

O FortiView inclui os seguintes painéis predefinidos que resumem diferentes aspectos da rede:
* A seção de Ameaças exibe as principais ameaças, um mapa global mostrando os destinos do tráfego e muito mais.
* O sistema de tráfego exibe a origem do tráfego, o destino, as ações aplicadas às políticas de tráfego e muito mais.
* Shadow IT mostra aplicativos em nuvem e usuários da nuvem.
* A seção Aplicativos e Sites mostra os principais aplicativos, domínios, categorias de sites e usuários que mais navegam pelos sites.
* A VPN exibe usuários de conexão discada SSL e IPsec, bem como usuários de VPN site-to-site.
* O sistema exibe logins do FortiAnalyzer, eventos do sistema e uso de recursos.

O FortiView permite o uso de múltiplos filtros, possibilitando restringir a visualização a um período específico, por ID de usuário ou endereço IP Local, por aplicativo e muito mais. Você também pode usar múltiplos filtros para investigar a atividade de tráfego, como uploads e downloads de usuários ou vídeos assistidos no Youtube, em um grupo de usuários de toda a rede ou para usuários individuais. É possível exportar informações do FortiView como um arquivo PDF ou criar um gráfico para usar em relatórios. 

# Exemplo de uso de informação em Widget

<img width="1389" height="784" alt="image" src="https://github.com/user-attachments/assets/020b6eb5-713a-4588-ac59-d13c5ecd2c04" />

A partir das informações exibidas em um widget, você pode encontrar mais detalhes sobre uma entrada específica. 

O widget Principais Ameaças exibe as 100 principais ameaças. A ameaça no topo da lista é um evento IPS como uma pontuação de ameaça muito alta. Se clicar duas vezes nessa entrada, verá mais detalhes, como os endereços IP de origem relacionados a esse tráfego. Neste exemplo, apenas um endereço IP é listado, indicando que este host provavelmente foi comprometido e o analista deve relatar as descobertas. As ameaças IPs listadas no widget também incluem os números de identificação de vulnerabilidades e exposições comuns (CVE) correspondentes, com hiperlinks nos quais você pode clicar para ler mais sobre os vetores de ataque, o impacto sugerido, o software vulnerável e muito mais.

A principal fonte da lista é um host que está acessando a rede por meio de VPN SSL, com uma pontuação de ameaça muito alta.
