Visualização de Logs

<img width="860" height="496" alt="image" src="https://github.com/user-attachments/assets/b030ab40-7871-47d8-9204-83abbfca92c8" />

A visualização de logs permite visualizar todos os tipos de logs recebidos pelo FortiAnalyer em formato normalizado. 
Quando o ADOM é habilitado, as informações de registros de logs para cada ADOM é mostrada. 


Pode-se escolher visualizar logs de dispositivos específicos, grupos Fabric, ou grupos de logs. 
Um grupo de log é um grupo de dispositivos alocados em um único objeto único. Os grupos de logs são virtuais, portanto, não possuem bancos de dados SQL nem ocupam espaço em disco adicional. 
Pode-se visualizar os logs gerados pelo FortiAnalyzer, os arquivos de log atuais e quaisquer arquivos de log rotativos. Você também pode importar logs de outros dispositivos FortiAnalyzer.

As opções disponíveis na visualização de logs variam dependendo do tipo de ADOM que é criado. Na imagem acima o ADOM é Fabric.

# Visualização de Logs no Fortigate

<img width="766" height="427" alt="image" src="https://github.com/user-attachments/assets/bc3ceba4-77eb-4223-8160-eabc1879681a" />

Logs de tráfego permitem visualizar o tráfego que atravessa os firewalls, incluindo detalhes como origem, destino, serviço, ação e muito mais. Os registros de segurança permitem visualizar registros relacionados à inspeção de gerenciamento unificado de ameaças (UTM). Os registros de eventos geralmente estão relacionados às operações do sistema de firewall. 

Pode ser selecionar o subtipo de log de segurança ou de eventos que deseja visualizar ou acessar um painel de resumo para obter uma visão geral. 

# Visualizando os logs do Fortigate 

<img width="764" height="438" alt="image" src="https://github.com/user-attachments/assets/3181d2c3-773c-4d34-8f8c-ccb3cd5e6166" />

Pode-se monitorar todas as permissões de segurança e tipos de logs de eventos dos seus respectivos dashboards. Os dashboards usam vários widgets para exibir os principais logs de cada subtipo. Pode-se adicionar ou remover widgets conforme necessário. Dashboards resumidos usam múltiplos widgets simultaneamente, então deve-se assegurar que sejam configurados corretamente. 

Pode-se clicar nos links fornecidos nas dashboards para acessar registros específicos. Ao fazer isso, será direcionado para a seção do subtipo de registro específico, com os filtros apropriados aplicados para pesquisar as entradas de registro desejadas.
