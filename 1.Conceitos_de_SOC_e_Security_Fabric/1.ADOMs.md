# ADOMs

<img width="687" height="394" alt="image" src="https://github.com/user-attachments/assets/9e8b5ca3-e323-4062-b7c2-ba6d4b48cde2" />

ADOMs permite agrupar disposisitivos para monitoramento e gerenciamento eficientes. Por exemplo, administradores podem gerenciar dispositivos baseados em suas localizações geográficas ou setor de negócio. 

O propósito primário dos ADOMs:

* Para dividir a administração dos dispositivos pelo ADOM e controlar o acesso administrativo. Se a rede usa VDOMs, ADOMs podem limitar ainda mais o acesso aos dados do VDOM em um dispositivo específico.
* Para gerenciar política de dados e alocação de espaço de disco de forma eficiente que são executados pelo ADOM.

ADOMs não são permtidos por default e podem ser configurados somente por um administador default.

Todos dispositivos da Fortinet que são parte do Security Fabric podem ser atribuidos para o ADOM do Fabric Type, permitindo o processamento rápido e correlação de logs.
